Bugs and risks recorded for this version

- Earth Engine authentication now works locally after setting `SSL_CERT_FILE` from `certifi`, but the authenticated account or project is still not enabled for Earth Engine use.
- Earth Engine currently returns: `Not signed up for Earth Engine or project is not registered`.
- Biomass carbon density is around 2010, while Sentinel-2 and Dynamic World are used for 2020-2021 study years, so temporal mismatch is a known modeling limitation.
- The first script version uses a GeoJSON file for custom polygons rather than a full interactive polygon-capture app.
- The yearly animation is a modeled estimate and should be treated as exploratory.
- The original `geemap.foliumap` import path failed in this environment with a `xyz_to_folium` basemap error, so the script now uses direct Folium layer construction instead.
